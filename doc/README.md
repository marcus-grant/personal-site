# personal-site documentation

Index of project documentation.
Documentation lives under topic-area subdirectories.
Each subdirectory has its own README acting as the entry point for that area.

## Topics

- [`design/`](./design/README.md)
  - Architecture, content contract, IndieWeb implementation, CSS posture.
  - What this renderer is making.
- [`usage/`](./usage/README.md)
  - sync-content script, 11ty config, deploy pipeline, conventions.
  - How to run and develop it.

## Reading order

Start with `design/` for context on what the renderer is and how it fits
the wider personal-site ecosystem.
Then `usage/` for the operational side once the design context lands.
